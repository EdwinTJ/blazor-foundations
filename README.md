# Blazor Learning Lab

A structured collection of small projects designed to learn and demonstrate core concepts of **Blazor**, **.NET**, and **C#**.

## Overview

This repository is a hands-on learning journey focused on building practical skills in modern web development using Blazor. Each project is intentionally small and focused, allowing concepts to be learned step-by-step and reinforced through repetition.

The goal is not just to understand syntax, but to build a strong foundation in:

* Component-based architecture
* State management
* Event handling
* Forms and validation
* Dependency injection
* API integration

## Structure

The solution contains multiple projects, each focused on a specific concept. Projects are organized in a progressive order to reflect increasing complexity and understanding.

## Learning Approach

Each project is built with the following principles:

* Keep it simple and focused
* Reinforce core concepts through repetition
* Gradually introduce more advanced patterns
* Emphasize clean and readable code

## Tech Stack

* Blazor
* .NET
* C#
* Razor Components
